# Password Generator

A simple password generator web app that creates multiple strong passwords based on a user-provided length.

## Features

- Generates 4 unique strong passwords at once
- Supports uppercase, lowercase, numbers, and symbols
- Validates password length input
- Responsive layout for smaller screens

## Usage

1. Open `index.html` in a browser.
2. Enter a password length between 4 and 128.
3. Click the **Generate Passwords** button.
4. Copy any generated password from the results.

## Files

- `index.html` — main page markup
- `style.css` — responsive styling and layout
- `script.js` — password generation logic

## Notes

This app is great for quickly creating random passwords for accounts or test data. It does not store passwords and generates each password locally in the browser.
